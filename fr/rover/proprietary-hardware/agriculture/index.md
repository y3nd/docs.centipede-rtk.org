# Utiliser Centipede-RTK sur du matériel agricole

Ce travail de compatibilité a été réalisé par la chaire AgroTIC de SupAgro avec un groupe d'agriculteurs testeurs. Le tableau présente un extrait d'une liste non exhaustive de tests de compatibilité réalisés avec le réseau Centipede-RTK par des utilisateurs travaillant en viticulture. Il recense les systèmes de guidage automatique qui ont fonctionné dans différentes configurations. La Chambre d'agriculture de la Somme a également mené un travail similaire avec des utilisateurs travaillant sur des cultures industrielles.
