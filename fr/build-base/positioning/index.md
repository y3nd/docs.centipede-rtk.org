# Calculer la position de la base

Nous allons maintenant déterminer la position de la base aussi précisément que possible.

Votre base servira de référentiel pour vous, mais aussi pour toute personne située dans sa zone de fonctionnement. Il est essentiel que sa position soit correcte et très précise afin que chacun puisse travailler avec le même référentiel géographique, quelle que soit la base du réseau à laquelle nous sommes connectés.

Pour cela, nous allons convertir un fichier journal de 24 heures contenant les données satellites reçues par votre base GNSS en un fichier [RINEX](http://rgp.ign.fr/DONNEES/format/rinex.php), puis le transmettre à un service de calcul de positionnement en ligne afin d'obtenir les coordonnées précises (latitude, longitude et altitude) de votre base RTK.
