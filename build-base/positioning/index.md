# Calculating the Base Position

We will now determine the position of the base as precisely as possible.

Your base will serve as a reference frame for you, but also for anyone within its area of operation. It is essential that its position is correct and highly precise so that everyone can work using the same geographic reference frame, regardless of which base in the network we are connected to.

To do this, we will convert a 24-hour log file of satellite data received by your GNSS Base into a [RINEX](http://rgp.ign.fr/DONNEES/format/rinex.php) file, then transfer it to an online positioning calculation service in order to obtain the precise coordinates (latitude, longitude, and elevation) of your RTK base.
